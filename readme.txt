A lightweight plugin to easily transform and brand your WordPress login screen with just one click, fully compatible with all themes and plugins.

== Description ==  
**Simple Login Screen for WordPress** allows you to effortlessly change your WordPress login page to match your brand or site theme. With just one click, you can apply a new login design and customize it further with premium features.

This plugin integrates seamlessly with WordPress's core CSS, ensuring full compatibility with any theme or plugin, without slowing down your site. 

### Key Features:
* Customize the login page to reflect your brand or site theme.
* One-click design change.
* Integrated with core WordPress CSS for smooth performance.
* Fully compatible with all themes and plugins.
* Provide a fresh new look to your site members.
* Pro version features: customize font sizes, colors, and background images.

== Installation ==  
1. Upload the plugin directory to `/wp-content/plugins/`  
2. Activate the plugin through the ‘Plugins’ menu in WordPress  
3. Navigate to **Settings > WP Login** to configure the options

== Frequently Asked Questions ==

= Can I change the login logo? =  
Yes, you can upload your own logo or use text instead of the logo.

= Is the plugin compatible with all themes and plugins? =  
Yes, the plugin is designed to work with any theme or plugin, without conflicts.
